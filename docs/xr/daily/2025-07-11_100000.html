NeuralOS – AI가 대화하며 실시간 UI 생성하는 기술

linkedin.com | 20250711
  • AI가 사용자와 대화하면서, 필요한 화면을 즉석에서 만들어주는 기술(NeuralOS)이 공개됨.
  • 마우스·키보드 입력에 따라어떤 상태에 있는지를 RNN으로 추적하고,그 상태에 맞는 화면을 이미지로 생성함.- 예: “이메일 보여줘” → 메일 목록 그림 생성- “PDF 첨부 열어줘” → 오른쪽에 미리보기 이미지 생성- Ubuntu OS의 UI 동작 수천 개를 학습하여 상태 전이까지 예측함
  • 디퓨전 모델 기반으로 픽셀 단위 UI를 직접 생성하며, 실험에서는 실제 클릭과 전환도 가능했음.다만프레임 속도는 1~2fps 수준으로 낮고,고성능 GPU(H100) 필요등 현실적 제약이 존재함.
  • 기존 OS처럼 정해진 앱·버튼 없이도 AI가 사용자의 의도만 이해하면그때그때 필요한 인터페이스를 구성가능함.→AI 비서·XR 기기·챗봇용 OS에서 활용 가치 매우 큼
  • AI와 대화하면서 필요한 작업 흐름을 그대로 시각화할 수 있어,향후 초개인화 UI 또는 "AgentOS"의 기반 기술로 평가됨.
